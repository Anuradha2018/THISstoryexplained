Code explaining THIS key word; For Output check console.

This refers to the object that is executing the current function

CASE 1- FOR METHODS:if a function is part of the object we call it a method. => 'This' in this case refers to the object.

CASE 2- FOR FUNCTIONS:if it is a regular function that means its not part of the object => 'This' refers to window object (Browsers) or global object (Node)
